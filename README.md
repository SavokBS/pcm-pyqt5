# pcm-pyqt5
Python Character Moving now in PyQt5!

PCM PyQt5 (Python Character Moving Lab) - a project which essence is to show what PyQt5 is capable of these days and train your own skills in it.

Abilities:
- Walking
- Shooting
- Building *NEW!*
- Debug menu *NEW!*
- Chat (better design)
